Welcome to PopIt

PopIt makes it simple to start and maintain a public database of people, organizations and positions.

Creation of new sites is currently disabled. We're making some fairly big changes to the codebase and don't want people to have the site break under them.

These changes will include making the API much simpler and switching it to use the Popolo standard for data validation. We'll also be making it much easier to use PopIt in your own website by providing code for use in frameworks like Django.

If you want to be informed when we re-enable site creation please give us your email address.
