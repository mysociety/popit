Data Import

PopIt will let you enter and access data easily from your code. It will also let you upload existing data to quickly populate a new site.

Migration Tool

If you have been collecting information on people in a spreadsheet, this web based tool will let you quickly upload it all into PopIt rather than entering it manually.

To use it, simply go to /migration in your PopIt instance and follow the instructions.

API

The API will let you access and modify all the data in PopIt, as well as storing your own custom data. Please see the API docs for more details.

We use the API from the editing interface on the website to be sure that all interactions with PopIt can be achieved through the API. It is not an afterthought - it is a core piece of the site.

Improving the API is an ongoing task with several open issues. If you have particular suggestions or questions please let us know on the poplus mailing list.
